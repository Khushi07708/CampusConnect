# 🎓 CampusConnect – Full-Stack Student Community Platform

CampusConnect is a full-stack web application designed to help college students access essential campus services from one platform. It provides features such as user authentication, notes sharing, a book marketplace, food services, room finder, and a contact module.

## 🚀 Live Demo

🌐 Frontend: https://campusconnect-project.netlify.app/

⚙️ Backend: https://campusconnect-dwrq.onrender.com/

## ✨ Features

- 🔐 User Registration & Login (JWT Authentication)
- 📚 Notes Sharing
- 📖 Book Marketplace
- 🍱 Food Services
- 🏠 Room Finder
- 📩 Contact Form
- ☁️ MongoDB Atlas Database
- 🌍 Fully Deployed Frontend & Backend

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Khushi07708/CampusConnect
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Create a `.env` file

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

### Start the Backend

```bash
npm start
```

### Run the Frontend

Open the `index.html` file using **Live Server** in VS Code.
